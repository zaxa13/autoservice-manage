# Обязательные скрипты

Все скрипты запускаются из папки `backend/` с активированным виртуальным окружением.

```bash
cd backend
source venv/bin/activate  # или: venv\Scripts\activate на Windows
```

---

## 1. Первичная настройка базы данных

```bash
alembic upgrade head
```

Применяет все миграции. Запускать при первом запуске и после каждого обновления.

---

## 2. Создание администратора

```bash
python scripts/create_admin.py
```

Создаёт первого пользователя с правами администратора. Запускать один раз после первого запуска.

---

## 3. Каталог марок и моделей автомобилей

```bash
python scripts/seed_car_catalog.py
```

Заполняет справочник марок и моделей (~30 марок, популярные в России). Запускать один раз.

---

## 4. Базовый каталог работ автосервиса

```bash
python scripts/seed_works_catalog.py
```

Заполняет каталог типовых работ (~259 позиций по 17 категориям):
- Диагностика, Двигатель, КПП / Трансмиссия
- Подвеска, Тормоза, Электрика
- Охлаждение, Топливная система, Выхлоп
- Климат / Кондиционер, ТО
- Кузов, Покраска, Шиномонтаж, Стёкла, Прочее

Скрипт проверяет дубликаты по названию и спрашивает подтверждение, если работы уже есть. Запускать один раз.

---

## Порядок запуска при первом развёртывании

```bash
alembic upgrade head
python scripts/create_admin.py
python scripts/seed_car_catalog.py
python scripts/seed_works_catalog.py
```
